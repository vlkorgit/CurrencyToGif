# CurrencyToGif

Используемые технологии:
- Srpingboot
- Feign
- JUnit
- Wiremock
- Lombok
- Gradle

Для сборки и запуска тестов
`gradle build`

Для запуска
 `gradle bootRun`
 
 Для сборки через IDE должен быть установлен lombok plugin

[Образ на докерхабе](https://hub.docker.com/repository/docker/vlkor/currency_to_gif)
